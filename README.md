# HBR Haber - Basit PHP Haber Sitesi


Statik HTML demo sürümü. PHP/SQLite bağımlılığı kaldırıldı, site statik dosyalardan oluşur.

Kullanım:

1. Tüm dosyaları hosting hesabınızın `public_html` veya web köküne yükleyin.
2. `index.html` ana sayfa, `article-1.html` örnek haber sayfasıdır. Yeni haber eklemek için yeni `.html` dosyaları oluşturun.

Notlar:
- Eğer dinamik bir admin ve veritabanı isterseniz, projeyi MySQL/PHP şeklinde yeniden yapılandırabilirim.
